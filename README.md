# 🔐 Mx Crypto Tool

![Version](https://img.shields.io/badge/version-1.0-red)
![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20Linux-green)

> *"Cifra como mexa, seguro como debe ser"*

## 🚀 Características

- Cifrado AES-256-CBC
- Cifrado RSA (generación de llaves pública/privada)
- Generación de hashes (MD5, SHA1, SHA256, SHA512)
- Verificación de integridad de archivos
- Base64 encoding/decoding
- Interfaz de menú interactivo
- Banner bien perrón

## 📦 Instalación

### Linux
```bash
git clone https://github.com/Falconmx1/Mx-Crypto-Tool.git
cd Mx-Crypto-Tool
chmod +x install.sh
./install.sh

Windows

git clone https://github.com/Falconmx1/Mx-Crypto-Tool.git
cd Mx-Crypto-Tool
python mx_crypto.py

🎮 Uso

python mx_crypto.py

🧰 Requisitos

    Python 3.6+

    pip install cryptography pycryptodome

👊 Comandos del menú

   1- Cifrar texto (AES)

   2- Descifrar texto (AES)

   3- Generar llaves RSA

   4- Cifrar con RSA

   5- Descifrar con RSA

   6- Generar hash de texto

   7- Hash de archivo

   8- Base64 encode/decode

   9- Salir
